Principais Formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito usabdo a tag <b> (não semântica).

Nesta frasem tenho um termo em destaque usando a tag <strong>

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Esamos criando um texto grande e um texto pequeno neste parágrafo

Texto Deletado

Podemos marcar um texto como excluído, para indicar que deve ser lido, mas não considerado, com a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins>

Existe também o sublinhado, utilizado com a tag <u> (não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3, use-se a tag <sup>

Texto Subscrito

Para inserir coisas do tipo H2O, usa-se a tag <sub>
